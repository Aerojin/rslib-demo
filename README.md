# rslib-demo

组件功能描述

## Install

```bash
$ npm i rslib-demo --save
```

## Usage

```jsx
import RslibDemo from 'rslib-demo';
```


## API

| 参数      | 说明                        | 类型                                | 默认值                        | 是否必填 |
| --------- | -------------------------- | -----------------------------------| ---------------------------- | ------- |
| type      | 按钮的类型                  | 'primary'                           | 'normal'                     |         |
| size      | 按钮的尺寸                  | ButtonSize                          | 'medium'                     |         |
